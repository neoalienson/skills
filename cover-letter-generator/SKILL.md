# Cover Letter Generator

A skill for generating personalized cover letters based on job descriptions and user's resume/skills, with PDF export functionality.

## Description

This skill allows users to:
- Generate personalized cover letters based on job descriptions
- Customize cover letters using their resume and skills
- Export cover letters as PDF documents
- Store generated cover letters locally

## Usage

When the user wants to generate a cover letter:
1. Parse the job description provided by the user
2. Retrieve user's resume/skills from stored data
3. Generate a personalized cover letter using AI
4. Format the cover letter appropriately
5. Export as PDF and save to local storage

## Implementation

The skill will use a template-based approach combined with AI to generate personalized cover letters that highlight relevant skills and experiences from the user's resume that match the job requirements.

## Examples

When the user says "Generate a cover letter for this job: [job description]":
- Parse the job description
- Retrieve user's resume/skills
- Generate a personalized cover letter
- Save as PDF with appropriate naming convention
